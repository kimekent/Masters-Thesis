Key: Weiterbildung MAS Site erstellen
seitentypen>template_weiterbildung
Achtung: Seiten nie kopieren oder duplizieren.
 
1) Im Ribbon-Menü den Befehl Home > Insert Weiterbildung wählen.
 

 
 
2) Weiterbildung MAS Site wählen, der Seite einen Namen geben (Item Name) und Insert klicken.
 

 
 
3) Es werden automatisch zwei Seiten erstellt. Zum einen die neu erstellte MAS Hauptseite und dann noch eine dazugehörige Unterseite "Module".
 
 
Die Seite mit den folgenden Elemente wird automatisch erstellt. Anschliessend können die einzelnen Elemente bearbeitet werden.
 
Hinweis: Die Weiterbildungs-Elemente können auch im Backend, über das Ribbon Menü > Campus > Backend bearbeitet werden.
 
Content Container:
1.Page Intro (Titel, Titel 2, Intro Text) >> Bearbeiten (Wichtig für die Google Suche)
2.Social Buttons (können nicht editiert werden)
3.ABToggleText >> Bearbeiten (Titel, Headline, Text)
4.Factsliste Weiterbildung >> Bearbeiten (Programmart, Dauer Zahl, Dauer Einheit, ICS-Datei, Leitung, Durchführungsort, Unterrichtssprache, Anbieter, Semester, Evento Anlass ID)
5.MAS Themen Zuordnung
 
Sidebar:
6.Seitennavigation (kann nicht editiert werden)
7.Kontaktperson (Manuell) >> Bearbeiten.
8.Info Veranstaltungen Aus- /Weiterbildungen: Bei Informations Event Seiten können Aus- und Weiterbildungsseiten verknüpft werden.
 
Hinweis: Wenn nötig, können auch noch weitere Elemente hinzugefügt werden.
 

 
Die Seite mit den folgenden Elemente wird automatisch erstellt. Anschliessend können die einzelnen Elemente bearbeitet werden.
 
Content Container:
9.Page Intro (Titel, Titel 2, Intro Text) >> Bearbeiten (Wichtig für die Google Suche)
10.WB Modul Generator >> Bearbeiten
 
Im Bereich 10 (WB Modul Generator) - in den vier dafür vorgesehenen Felder, können Einleitungstexte für die Rubriken Pflichtmodul, Wahlpflichtmodul, Wahlmodul und Abschlussarbeit definiert werden.
 
Sidebar:
11.Seitennavigation (kann nicht editiert werden)
 
Hinweis: Wenn nötig, können auch noch weitere Elemente hinzugefügt werden.

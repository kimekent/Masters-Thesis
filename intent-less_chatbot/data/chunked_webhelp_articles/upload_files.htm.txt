Key: Upload Files via Media Library

Diese Anleitung beschreibt, wie Datein in die Media Library geladen werden können. Zudem werden die Beschriftungsfelder beschrieben und aufgezeigt wie die Darstellungspriorisierung bei den Dateien aussieht. Wichtig ist hier auch der Abschnitt zu Zweisprahige Medien da er verrät wie man eine englische Version einer Datei hochladen kann und dadurch die Meldung "Only in German" verhindern kann.

Pfad im Page-Editor: Ribbon-Menü > Campus > Media Library
 
•Bilder: Medien-Bibliothek/Campus/Common/Images
•PDFs und Dateien: Medien-Bibliothek/Campus/Common/Files
   
Upload via Button "Upload files":
1.Im Ribbon-Menü Campus > Media Library wählen.
 
2.In der Media Library im Item Tree der entsprechenden Ordner für deinen Speicherort auswählen. Wenn kein Ordner ausgewählt wird, erscheint eine Fehlermeldung, die das Speichern der Datei oder des Bildes verhindert.
 
Hinweis: Für eine neue Galerie einen neuen Ordner erstellen:
Rechtsklick auf einen bestehenden Ordner, dann Insert > Media Folder klicken und den Ordner benennen.
 
3.Im Reiter Folder auf das Upload-Files-Icon klicken.
 
4.Gewünschte Dateien im Explorer-Fenster auswählen, z.B. Bilder und auf Öffnen klicken. Die Bilder werden hoch geladen.
 
Hinweis: Mit gedrückter Ctrl-Taste können mehrere Files ausgewählt werden.
 
5.Die Files wurden in den ausgewählten Ordner (hier der Ordner ta) hoch geladen. Im rechten Bereich Media werden die Bildinhalte mit einem Warnhinweis dargestellt. Dies, weil bei den Bilder noch die Alt-Texte fehlen und ausgefüllt werden müssen.
 
6.Links im Item Tree auf das jeweilige Bild klicken und anschliessend im Content Bereich auf der rechten Seite den Alt-Text eingeben. Dieser Text sollte das Bild beschreiben. Die Eingaben speichern.
 
Hinweis Titel-Feld (siehe folgendes Beispiel): Wird ein Bild oder ein PDF z.B. in einer Link und Download Liste verwendet, sollte auch noch der Titel eingetragen werden. Bei einem PDF kann zudem auch noch eine Beschreibung eingegeben werden.
Vorteil: Der Titel wird als Linktext verwendet. Titel und Beschreibung sind zweisprachige Felder.
Die Link description (beim setzen von einem Link) ist hingegen nur einsprachig und sollte in diesem Fall nicht verwendet werden.
 
Priorisierung der Datenfelder und deren Anzeige-Reihenfolge: 
1.Link Description: Wird eine Link-Description beim Link selber hinterlegt (sollte möglichst vermieden werden), wird immer diese Eingabe angezeigt (gewinnt immer).
2.Titel: Ist keine Link-Description vorhanden (Normalfall), dafür wurde aber ein PDF-Titel eingegeben, wird dieser angezeigt.
3.Dateiname / Display Name: Wurde keine Link-Description und kein Titel erfasst, wird der Dateiname / Display Name angezeigt.
 
PDF-Beschreibung:
Hinweis: Die Beschreibung zum PDF wird im Web nicht mehr angezeigt. Früher wurde diese Information bei der Suchresultaten der Globalen Suche mit angezeigt. Daher haben diese beiden Felder keine Relevanz mehr.

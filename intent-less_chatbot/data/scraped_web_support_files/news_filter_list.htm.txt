Key: News Filter List konfigurieren
Navigation:  Seitenelemente > Template Container > Such- und Filterlisten > Filter Konfigurationen

In dieser Anleitung wird erklärt, wie eine News Filter Liste konfiguriert werden kann. Es wird aufgelistet welche Einstellungen vorgenommen werden können udn erklärt was Fixe und was vordevinierte Filter sind.

Siehe auch: News erfassen (https://applsupport.hslu.ch/webhelp/hm_news_einfuegen.htm), News Teaser (https://applsupport.hslu.ch/webhelp/hm_multi_news_teaser_einfuegen.htm), Multi News Teaser (https://applsupport.hslu.ch/webhelp/index.html?hm_multi_news_teaser_einfuegen.htm)

Die Filterliste auswählen und auf der folgenden drei Buttons klicken:
 
• Filterliste bearbeiten
• vorselektierte Filter bearbeiten
• fixe Filter bearbeiten
 
Filterliste bearbeiten:
•Filter ausblenden: Wird dies aktiviert, verschwinden für die User die Filtermöglichkeiten.
•Kein Resultat Text: Der Text, welcher in diesem Feld eingetragen ist, wird angezeigt, wenn keine Resultate gefunden wurden.
•Filter standartmässig ausgeklappt: Seit November 2015 werden die Filter standartmässig zugeklappt dargestellt. Dies spart Platz in der Browser-Ansicht. Wird diese Check-Box hingegen aktiviert, werden die Filter wieder aufgeklappt dargestellt.

Vorselektierte Filter bearbeiten: 
•OE vorselektiert: OE vorselektieren. User behält die Möglichkeit OE zu wechseln.
•NewsFilterListPreSelectedCategory: Kategorie vorselektieren. User behält die Möglichkeit Kategorie zu wechseln.
•Themen vorselektiert: Thema vorselektieren. User behält die Möglichkeit Thema zu wechseln.
 
1.Das gewünschte Element in der linken Spalte All doppelklicken oder mittels der Links- und Rechts-Pfeile auswählen (heisst in die Selected-Spalte verschieben).
2.OK klicken.
 
Wichtig: Speichern (ctrl + s), denn die Änderungen werden erst nach dem Speichern sichtbar!
 
fixe Filter bearbeiten: 
•OE fix: OE fix vordefinieren. User hat keine Möglichkeit OE zu wechseln.
•NewsFilterListFixCategory: Kategorie fix vordefinieren. User hat keine Möglichkeit, Kategorie zu wechseln.
•Themen fix: Themen fix vordefinieren. User hat keine Möglichkeit Thema zu wechseln.
•NewsFilterListFixLeistungsAuftrag: News-Mitteilung einem bestimmten Leistungsauftrag zuordnen.
 
1.Das gewünschte Element in der linken Spalte All doppelklicken oder mittels der Links- und Rechts-Pfeile auswählen (heisst in die Selected-Spalte verschieben).
2.OK klicken.
 
Wichtig: Speichern (ctrl + s), denn die Änderungen werden erst nach dem Speichern sichtbar!

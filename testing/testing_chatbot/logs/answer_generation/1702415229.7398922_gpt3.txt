Q: Ich habe bei folgender Website www.hslu.ch/de-ch/wirtschaft/weiterbildung/weiterbildungskurse/ibr/managementrefresher/ im Sitecore unter "Publish Options" die Publizierung auf heute 18.3. gesetzt, doch die Website ist heute nicht online. Könntet ihr dies bitte fuer mich vornehmen?

 
A: Wenn eine Seite nicht erscheint kann das an den Publish Einstellungen liegen: 
Im Experience Editor kann auf der Seite über die den Befehl "Campus > Publish Options" im Ribbon-Menü kontrolliert werden, ob und wann Inhalte publiziert werden.

Im Feld "Publish" definiert die Redaktion, wann ein nicht veröffentlichter Inhalt publiziert wird. Achtung: Die Seite wird immer auf den nächstfolgenden Veröffentlichungszeitpunkt live geschalten (entweder um 12:05 oder 00:05).

Im Feld "Unpublish" definiert die Redaktion, wann ein veröffentlichter Inhalt zurückgezogen wird.

Ist die Checkbox "Never publish" angekreuzt, wird ein Inhalt nicht publiziert.
Publish / Unpublish kann auch bei einzelnen Teaser oder auf Files in der Media Library angewandt werden.

Ein weiterer Grun könnte an den Sprachversionen liegen. Seiten wie auch Seitenelemente müssen immer auf deutsch vorhanden sein. Wenn ein Teaser in der deutschen Version gelöscht wird, wird er auch in der englischen gelöscht. Es gilt die Regel, keine Inhalte nur in englisch, da dies fuer SEO schädlich ist.

Q: Die Konferenz Schweizer Vermögensverwalter ist erstellt. Aber leider findet man die Konferenz ueber die Suchfunktion nicht. Dies passiert in letzter Zeit leider sehr häufig. Könnt Ihr dies bitte pruefen.
A: Wenn eine Seite über die Suche nicht gefunden werden kann, liegt es warscheinlih darasn, dass die Seite von der internen Meta-Suche ausgeschlossen wurde.
 Meta Daten können so bearbeitet werden:
1) Im Content-Tree zur entsprechenden Seite navigieren.

2) Rechts im Content-Bereich unter Site Meta Data können die Meta-Daten bearbeitet werden.

3) Nach Eingabe, die Daten mit ctrl + s sichern. 

Um eine Seite von der internen Suche ein- oder auszublenden muss das Häckchen in den Metadaten unter "Von interner Meta-Suche ausschliessen (wird für Events automatisch gesetzt)" gesetzt bzw. entfernt werden.
Diese Funktion kann manuell verwendet werden, um eine Seite von der internen Suche auszuschliessen. Bei abgelaufenen Events wird dieses Häkchen automatisch gesetzt. 
Mehr Infos dazu findest du [hier](https://applsupport.hslu.ch/webhelp/index.html?hm_site_meta_data_content_editor.htm).

Key: Person Filter List konfigurieren
Navigation:  Seitenelemente > Template Container > Such- und Filterlisten > Filter Konfigurationen >

In dieser Anleitung wird erklärt, wie eine Personen filterliste konfiguriert werden kann. Es wird aufgelistet welche Einstellungen vorgenommen werden können udn erklärt was Fixe und was vordevinierte Filter sind.

Optionen zugreifen:
Die Filterliste auswählen und auf der folgenden drei Buttons klicken:
 
• Filterliste bearbeiten
• vorselektierte Filter bearbeiten
• fixe Filter bearbeiten
 
Filterliste bearbeiten (welche Einstellungen können bei der Liste vorgenommen werden):
•Filter ausblenden: Wird dies aktiviert, verschwinden für die User die Filtermöglichkeiten.
•Kein Resultat Text: Der Text, welcher in diesem Feld eingetragen ist, wird angezeigt, wenn keine Resultate gefunden wurden.
•Filter standartmässig ausgeklappt: Seit November 2015 werden die Filter standartmässig zugeklappt dargestellt. Dies spart Platz in der Browser-Ansicht. Wird diese Check-Box hingegen aktiviert, werden die Filter wieder aufgeklappt dargestellt.

Vorselektierte Filter bearbeiten:
Filter vorselektiert: Die hier ausgewählten Filter sind Voreinstellungen. Die User können diese Voreinstellungen über steuern.
 
•OE vorselektiert: OE vorselektieren. User kann Studienort wechseln.
 
1.Das gewünschte Element in der linken Spalte All doppelklicken oder mittels der Links- und Rechts-Pfeile auswählen (heisst in die Selected-Spalte verschieben).
2.OK klicken.
 
Wichtig: Speichern (ctrl + s), denn die Änderungen werden erst nach dem Speichern sichtbar!
 
fixe Filter bearbeiten:
Filter fix: Die hier ausgewählten Filter sind fix. Die User können diese nicht beeinflussen.
 
•OE fix: OE fix bestimmen. User kann OE nicht wechseln.
 
1.Das gewünschte Element in der linken Spalte All doppelklicken oder mittels der Links- und Rechts-Pfeile auswählen (heisst in die Selected-Spalte verschieben).
2.OK klicken.
 
Wichtig: Speichern (ctrl + s), denn die Änderungen werden erst nach dem Speichern sichtbar!
